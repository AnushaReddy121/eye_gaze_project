# Eye Gaze Tracking System

An innovative accessibility project for physically handicapped users using webcam-based eye tracking.

### Features
- Tracks eye position using OpenCV
- Moves the mouse with gaze direction
- Blink or dwell for clicks
- Calibration system for better accuracy
