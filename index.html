Eye Gaze Project

This project uses Python scripts to track your eyes and control mouse clicks. You can run the scripts locally after setting up Python and the required packages.

Setup Instructions

  1. Clone or download this repository.
  2. Create a Python virtual environment: 
    python -m venv eye_env
  3. Activate the environment: 
    .\eye_env\Scripts\activate  (Windows PowerShell)
  4. Install dependencies: 
    pip install -r requirements.txt
  5. Download shape_predictor_68_face_landmarks.dat and place it in the project folder.

Available Scripts

calibrator.py

Calibrates the eye tracker for your screen. Follow on-screen points.

Run: python calibrator.py

camera_test.py

Tests webcam and displays live video feed with eye landmarks.

Run: python camera_test.py

clicker.py

Tracks eye blinks to simulate mouse clicks.

Run: python clicker.py

eye_gaze_opencv.py

Tracks eye gaze position and prints coordinates or moves pointer.

Run: python eye_gaze_opencv.py
