Eye Gaze Project

Welcome! This project uses Python scripts to detect eye movements and clicks based on blinking. You can download the scripts and run them locally in your Python environment.

Project Files

  • calibrator.py
  • camera_test.py
  • clicker.py
  • eye_gaze_opencv.py
  • README.md
  • requirements.txt

Instructions

  1. Clone or download this repository.
  2. Install the required dependencies using: 
    pip install -r requirements.txt
  3. Run the scripts in your Python environment.

Note

These scripts cannot run directly on GitHub Pages because it only serves static content. Run them locally using Python.
